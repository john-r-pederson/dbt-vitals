# Support

## Bugs and unexpected behaviour

Open a [GitHub Issue](https://github.com/Laskr/dbt-vitals/issues/new?template=bug_report.md) using the bug report template. Include the full error from the Actions log and your dbt-vitals version.

## Feature requests

Open a [GitHub Issue](https://github.com/Laskr/dbt-vitals/issues/new?template=feature_request.md) using the feature request template.

## Questions and general discussion

Use [GitHub Discussions](https://github.com/Laskr/dbt-vitals/discussions) for setup questions, how-to questions, and anything that isn't a clear bug or feature request.

## Security vulnerabilities

Do not file a public issue. See [SECURITY.md](SECURITY.md) for the private disclosure process.
